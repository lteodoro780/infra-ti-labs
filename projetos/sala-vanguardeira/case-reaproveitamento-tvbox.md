# Case técnico — reaproveitamento de TV Box

## Resumo

Projeto de reaproveitamento de TV Boxes para criação de estações Linux leves com foco educacional.

## Objetivos

- reduzir custo;
- reaproveitar hardware;
- montar ambiente funcional;
- padronizar configuração;
- permitir uso em laboratório/sala inteligente.

## Atividades técnicas

- identificação de hardware;
- teste de imagem Linux;
- boot por cartão SD;
- configuração inicial;
- padronização de interface;
- testes de uso;
- documentação do processo.

## Competências demonstradas

- Linux embarcado;
- troubleshooting;
- redes;
- manutenção;
- documentação técnica;
- sustentabilidade em TI;
- suporte ao usuário.
