# Estudo de Caso Sanitizado - Reaproveitamento de Hardware

Estudo genérico sobre reaproveitamento de equipamentos de baixo custo para criação de ambiente computacional educacional.

## Objetivo

Documentar aprendizados técnicos sobre:

- instalação de Linux em hardware limitado;
- padronização de ambiente;
- manutenção simplificada;
- sustentabilidade e reaproveitamento tecnológico.

## Segurança

Não incluir fotos, nomes institucionais, documentos internos, listas de presença ou qualquer dado não público.
