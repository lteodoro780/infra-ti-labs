# TV Box Linux Lab

Estudo sobre reaproveitamento de TV Boxes como computadores Linux de baixo custo.

## Temas

- Armbian
- RK322x
- Boot por cartão SD
- Multitool
- Limitações de hardware
- Uso educacional e sustentável
