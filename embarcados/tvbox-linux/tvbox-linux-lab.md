# TV Box Linux Lab

## Objetivo

Reaproveitar TV Boxes como computadores Linux leves para uso educacional, laboratório ou terminais de baixo custo.

## Exemplos de hardware

- TV Box com SoC RK322x;
- 1 GB de RAM;
- armazenamento limitado;
- boot via cartão SD;
- uso com Armbian ou sistemas compatíveis.

## Ferramentas

- Armbian;
- Multitool;
- imagem Linux compatível;
- cartão microSD;
- teclado, mouse e monitor.

## Limitações

- pouca RAM;
- armazenamento lento;
- Wi-Fi nem sempre funciona;
- DTB pode exigir testes;
- aceleração gráfica limitada.

## Boas aplicações

- laboratório educacional;
- terminal leve;
- navegação simples;
- acesso a sistemas web;
- reaproveitamento sustentável.
