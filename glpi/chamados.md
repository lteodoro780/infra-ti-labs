# Chamados GLPI

## Estados comuns

| Estado | Significado |
|---|---|
| Novo | Chamado recém-aberto |
| Em atendimento | Técnico assumiu ou iniciou tratativa |
| Pendente | Aguardando usuário, fornecedor ou ação externa |
| Solucionado | Solução registrada |
| Fechado | Chamado encerrado |

## Boa prática de solução

Uma solução deve conter:

- causa provável;
- ação executada;
- validação feita;
- orientação ao usuário, se aplicável.

Exemplo:

```text
Foi identificado problema de conectividade local. O cabo de rede foi reconectado e o link voltou a responder. Testes de ping e acesso ao sistema foram realizados com sucesso.
```
